# Git for Beginners

A simple Python project used to learn the basics of Git and GitHub.

## Project Structure

- `main.py` — Entry point that runs all calculator functions.
- `utils/calculator.py` — A basic calculator module with the following functions:
  - `add_nums` — Returns the sum of two numbers.
  - `divide_nums` — Returns the division of two numbers.
  - `subtract_nums` — Returns the difference of two numbers.
  - `power` — Returns a number raised to the power of another.
  - `multiply_nums` — Returns the product of two numbers.

## How to Run
`python main.py`
